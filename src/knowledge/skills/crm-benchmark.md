# CRM Benchmark

Aprender de todos, copiar de nenhum. Cada produto abaixo é referência em algo específico.

## Enterprise

### Salesforce Sales Cloud
- **Força**: metadata-driven, ecossistema (AppExchange), Einstein AI, Platform Events
- **Aprender**: modelo de segurança (profiles + permission sets + sharing rules), Flow Builder (automação declarativa)
- **Evitar**: complexidade de configuração, UI datada em módulos legados

### Microsoft Dynamics 365 Sales
- **Força**: integração Office/Teams, Copilot nativo, Dataverse
- **Aprender**: conversation intelligence, sales accelerator (workflow de sequência)
- **Evitar**: dependência de stack Microsoft

### SAP Sales Cloud
- **Força**: integração com SAP ERP, indústria
- **Aprender**: modelagem de account hierarchy pesada
- **Evitar**: UX corporativa fria

### Oracle CX Sales
- **Força**: enterprise puro, verticais (telecom, financeiro)
- **Aprender**: territory management robusto

## Mid-Market

### HubSpot CRM
- **Força**: onboarding fácil, marketing hub integrado, inbound methodology
- **Aprender**: associations (contact ↔ company ↔ deal), sequences, snippets
- **Evitar**: fica caro no mid-market conforme escala

### Zoho CRM
- **Força**: preço, Zia AI, ecossistema Zoho
- **Aprender**: canvas designer (customização de layout sem código)

### Pipedrive
- **Força**: pipeline visual, foco em atividade, simplicidade
- **Aprender**: activity-focused UX, insights minimalistas, quotas visuais

### Freshsales (Freshworks)
- **Força**: Freddy AI, integração com Freshdesk
- **Aprender**: 360° view integrando sales + support

### Copper CRM
- **Força**: integração nativa com Google Workspace
- **Aprender**: capturar dados de email/calendar sem esforço

## Modern SaaS (referência de UX)

### Attio
- **Força**: records flexíveis, colaboração ao vivo, datagrid poderoso
- **Aprender**: paradigma "records" (schemaless), lists (views salvas ricas), workflows visuais
- **Evitar**: complexidade para quem vem de CRM tradicional

### Folk
- **Força**: simplicidade, integração com email/LinkedIn, personal-first
- **Aprender**: enriquecimento zero-touch, magic imports

### Close CRM
- **Força**: inside sales (calling nativo), speed
- **Aprender**: power dialer, cadences focadas em call

### Monday CRM
- **Força**: flexibilidade visual, boards coloridos
- **Aprender**: mesma engine servindo CRM + projeto
- **Evitar**: excesso de personalização sem opinião

### ClickUp CRM
- **Força**: all-in-one (tasks + docs + CRM)
- **Evitar**: superfície demais, foco de menos

## Customer Success

### Gainsight
- **Força**: pioneiro do CS, health score maduro, playbooks
- **Aprender**: success plans, C360, rules engine

### Totango
- **Força**: SuccessBLOCs (templates de jornada)
- **Aprender**: composability

### Planhat
- **Força**: modern UI de CS, colaboração
- **Aprender**: portfolio view, workflows visuais

### ChurnZero
- **Força**: real-time alerts, in-app messages
- **Aprender**: gatilhos de intervenção proativa

### Vitally
- **Força**: automação profunda, integração com Slack
- **Aprender**: notion-like docs de conta

## Revenue Intelligence

### Gong
- **Força**: conversation intelligence (call recording + análise), deal warnings
- **Aprender**: "trackers" (detecção de menções), reality check em forecast

### Clari
- **Força**: forecast probabilístico, revenue platform completa
- **Aprender**: pulse survey de forecast, roll-up hierárquico

### People.ai
- **Força**: activity capture automático (email/calendar), enrichment
- **Aprender**: reduzir input manual a zero

### Aviso AI
- **Força**: predictive forecast, guided selling
- **Aprender**: guided workflow para deal em risco

## Sales Engagement / Cadence

### Outreach
- **Força**: sequences enterprise, analytics de cadence
- **Aprender**: A/B de sequence, opt-out global

### Salesloft
- **Força**: cadence + coaching + intelligence
- **Aprender**: rhythm (fila diária do rep), analytics de rep

### Apollo.io
- **Força**: dados + engagement combinados
- **Aprender**: search-driven prospecting

## Prospecting / Data

### ZoomInfo
- **Força**: maior base B2B, intent data
- **Aprender**: intent signals

### Cognism
- **Força**: dados EU-compliant (GDPR)
- **Aprender**: compliance-first data

### Lusha
- **Força**: leve, extensão browser
- **Aprender**: UX minimalista de enrichment

## Maps / Field Sales

### Badger Maps
- **Força**: rota otimizada para reps de campo
- **Aprender**: check-in, offline, agrupamento por cidade

### Map My Customers
- **Força**: visualização de território
- **Aprender**: heatmap de receita

### MapaLeads (Brasil)
- **Força**: dados CNPJ + CNAE brasileiros
- **Aprender**: prospecção por CNAE, integração com Receita Federal

## Como usar este benchmark
1. Nunca copiar visual/marca
2. Sempre citar internamente ("Attio faz X assim; nossa versão deve...")
3. Adaptar ao contexto Brasil / PME-mid mkt / integração ERP
4. Manter identidade USE CRM (Copiloto Comercial, não commodity)
